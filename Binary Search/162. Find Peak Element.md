# Problem 162. Find Peak Element
![Medium](https://img.shields.io/badge/Level-Medium-yellow?style=for-the-badge) [![Problem Link](https://img.shields.io/badge/LeetCode-Link-orange?style=for-the-badge&logo=leetcode)](https://leetcode.com/problems/find-peak-element/description/)

## ðŸ Python Solution
```python
class Solution:
  def findPeakElement(self, nums: list[int]) -> int:
    l = 0
    r = len(nums) - 1

    while l < r:
      m = (l + r) // 2
      if nums[m] >= nums[m + 1]:
        r = m
      else:
        l = m + 1

    return l
```

## ðŸ”¹ Approach Used

* **Binary Search on Answer**
* Use binary search to find a peak element.
* Compare the middle element with its right neighbor:

  * If `nums[mid] >= nums[mid + 1]`, a peak lies on the left (including mid).
  * Otherwise, a peak lies on the right.
* Narrow the search space until one index remains.
