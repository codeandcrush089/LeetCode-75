# Problem 162. Find Peak Element
![Medium](https://img.shields.io/badge/Level-Medium-yellow)

## Python Solution
```python
class Solution:
  def findPeakElement(self, nums: list[int]) -> int:
    l = 0
    r = len(nums) - 1

    while l < r:
      m = (l + r) // 2
      if nums[m] >= nums[m + 1]:
        r = m
      else:
        l = m + 1

    return l
```
