```python
# The guess API is already defined for you.
# @param num, your guess
# @return -1 if num is higher than the picked number
#          1 if num is lower than the picked number
#          otherwise return 0
# def guess(num: int) -> int:

class Solution:
  def guessNumber(self, n: int) -> int:
    l = 1
    r = n

    # Find the first guess number that >= the target number
    while l < r:
      m = (l + r) // 2
      if guess(m) <= 0:  # -1, 0
        r = m
      else:
        l = m + 1

    return l
```

## ðŸ”¹ Approach Used

* **Binary Search**
* Search within the range `1` to `n`.
* Use the `guess()` API feedback to decide the search direction:

  * If the guess is **too high or correct**, move left.
  * If the guess is **too low**, move right.
* Continue until the search range narrows to one number.
