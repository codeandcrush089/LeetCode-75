# Problem 714. Best Time to Buy and Sell Stock with Transaction Fee
![Medium](https://img.shields.io/badge/Level-Medium-yellow)

## Python Solution
```python
class Solution:
  def maxProfit(self, prices: list[int], fee: int) -> int:
    sell = 0
    hold = -math.inf

    for price in prices:
      sell = max(sell, hold + price)
      hold = max(hold, sell - price - fee)

    return sell
```

      hold = Math.max(hold, sell - price - fee);
    }

    return sell;
  }
}
