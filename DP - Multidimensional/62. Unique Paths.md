## ðŸ Python Solution
```python
class Solution:
  def uniquePaths(self, m: int, n: int) -> int:
    # dp[i][j] := the number of unique paths from (0, 0) to (i, j)
    dp = [[1] * n for _ in range(m)]

    for i in range(1, m):
      for j in range(1, n):
        dp[i][j] = dp[i - 1][j] + dp[i][j - 1]

    return dp[-1][-1]
```

class Solution {
  public int uniquePaths(int m, int n) {
    // dp[i][j] := the number of unique paths from (0, 0) to (i, j)
    int[][] dp = new int[m][n];
    Arrays.stream(dp).forEach(A -> Arrays.fill(A, 1));

    for (int i = 1; i < m; ++i)
      for (int j = 1; j < n; ++j)
        dp[i][j] = dp[i - 1][j] + dp[i][j - 1];

    return dp[m - 1][n - 1];
  }
}

## ðŸ”¹ Approach Used

* **Dynamic Programming (Grid DP)**
* Use a 2D DP table where `dp[i][j]` represents the number of unique paths to cell `(i, j)`.
* A cell can be reached either from the top or from the left.
* Fill the table iteratively using the relation
  `dp[i][j] = dp[i âˆ’ 1][j] + dp[i][j âˆ’ 1]`.
* The bottom-right cell contains the total number of unique paths.
