```python
class Solution:
  def deleteMiddle(self, head: ListNode | None) -> ListNode | None:
    dummy = ListNode(0, head)
    slow = dummy
    fast = dummy

    while fast.next and fast.next.next:
      slow = slow.next
      fast = fast.next.next

    # Delete the middle node.
    slow.next = slow.next.next
    return dummy.next
```

## ðŸ”¹ Approach Used

* **Two-Pointer (Slow & Fast) Technique**
* Use two pointers starting from a dummy node.
* Move `fast` two steps and `slow` one step at a time.
* When `fast` reaches the end, `slow` is just before the middle node.
* Delete the middle node by adjusting pointers.
