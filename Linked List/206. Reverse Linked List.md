# Problem 206: Reverse Linked List

```python
class Solution:
  def reverseList(self, head: ListNode | None) -> ListNode | None:
    if not head or not head.next:
      return head

    newHead = self.reverseList(head.next)
    head.next.next = head
    head.next = None
    return newHead
```

## ðŸ”¹ Approach Used

* **Recursive Linked List Reversal**
* Recursively reverse the sublist starting from the second node.
* Fix the links by making the next node point back to the current node.
* Set the current nodeâ€™s `next` to `None` to avoid cycles.
* Return the new head of the reversed list.
