# Problem 1318: Minimum Flips to Make a OR b Equal to c
![Medium](https://img.shields.io/badge/Level-Medium-yellow)

## Python Solution
```python
class Solution:
  def minFlips(self, a: int, b: int, c: int) -> int:
    MAX_BIT = 30
    ans = 0

    for i in range(MAX_BIT):
      if c >> i & 1:
        ans += (a >> i & 1) == 0 and (b >> i & 1) == 0
      else:  # (c >> i & 1) == 0
        ans += (a >> i & 1) + (b >> i & 1)

    return ans
```


## C++ Solution
```cpp
class Solution {
 public:
  int minFlips(int a, int b, int c) {
    constexpr int kMaxBit = 30;
    int ans = 0;

    for (int i = 0; i < kMaxBit; ++i)
      if (c >> i & 1)
        ans += (a >> i & 1) == 0 && (b >> i & 1) == 0;
      else  // (c >> i & 1) == 0
        ans += (a >> i & 1) + (b >> i & 1);

    return ans;
  }
};
```

## Java Solution
```java
class Solution {
  public int minFlips(int a, int b, int c) {
    final int MAX_BIT = 30;
    int ans = 0;

    for (int i = 0; i < MAX_BIT; ++i)
      if ((c >> i & 1) == 1)
        ans += ((a >> i & 1) == 0 && (b >> i & 1) == 0) ? 1 : 0;
      else // (c >> i & 1) == 0
        ans += ((a >> i & 1) == 1 ? 1 : 0) + ((b >> i & 1) == 1 ? 1 : 0);

    return ans;
  }
}
```
