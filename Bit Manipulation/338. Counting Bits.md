# Problem 338: Counting Bits
![Easy](https://img.shields.io/badge/Level-Easy-brightgreen)

## Python Solution
```python
class Solution:
  def countBits(self, n: int) -> list[int]:
    # f(i) := i's number of 1s in bitmask
    # f(i) = f(i / 2) + i % 2
    ans = [0] * (n + 1)

    for i in range(1, n + 1):
      ans[i] = ans[i // 2] + (i & 1)

    return ans
```

## C++ Solution
```cpp
class Solution {
 public:
  vector<int> countBits(int n) {
    // f(i) := i's number of 1s in bitmask
    // f(i) = f(i / 2) + i % 2
    vector<int> ans(n + 1);

    for (int i = 1; i <= n; ++i)

    return ans;
  }
};
```
