```python
class RecentCounter:
  def __init__(self):
    self.q = collections.deque()

  def ping(self, t: int) -> int:
    self.q.append(t)
    while self.q[0] < t - 3000:
      self.q.popleft()
    return len(self.q)
```

class RecentCounter {
  public int ping(int t) {
    q.offer(t);
    while (q.peek() < t - 3000)
      q.poll();
    return q.size();
  }

  private Queue<Integer> q = new LinkedList<>();
}

## ðŸ”¹ Approach Used

* **Sliding Window with Queue**
* Store all ping timestamps in a queue.
* For each new ping, remove timestamps older than `t âˆ’ 3000`.
* The remaining elements in the queue represent valid pings.
* Return the size of the queue.
