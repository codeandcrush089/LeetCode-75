# 933. Number of Recent Calls

## Python
```python
class RecentCounter:
  def __init__(self):
    self.q = collections.deque()

  def ping(self, t: int) -> int:
    self.q.append(t)
    while self.q[0] < t - 3000:
      self.q.popleft()
    return len(self.q)
```

## C++
```cpp
class RecentCounter {
 public:
  int ping(int t) {
    q.push(t);
    while (q.front() < t - 3000)
      q.pop();
    return q.size();
  }

 private:
  queue<int> q;
};
```

## Java
```java
class RecentCounter {
  public int ping(int t) {
    q.offer(t);
    while (q.peek() < t - 3000)
      q.poll();
    return q.size();
  }

  private Queue<Integer> q = new LinkedList<>();
}
```
